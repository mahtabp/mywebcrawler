# Web Crawler API

This API receives a string of keywords and does a search of Google. The result is then processed to return a string of numbers for the position(s) that the company URL is
found in Google search.

## Restrictions
Although it's suggested not to use regex for html parsing, we intentionally pracctice parsing html using regex instead of HTML agility.
